\doxysubsection{Detailed Description}
Contains the various flags that represent the status of the module. 

Those flags are useful since it indicates if a module was correctly initialized, in this way when the module must be unloaded, no deallocation is performed on modules that failed.

\item initialized\+: indicates that a group has loaded all of its structures

\item wake\+\_\+up\+\_\+flag\+: to implement
\item sysfs\+\_\+loaded\+: indicate that the \textquotesingle{}sysfs\textquotesingle{} interface is initialized
\item garbage\+\_\+collector\+\_\+disabled\+: specifies the status of the garbage collector
\item sysfs\+\_\+loaded\+: indicate that the \textquotesingle{}sysfs\textquotesingle{} interface is initialized
