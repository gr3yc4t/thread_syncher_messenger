\doxysubsection{Detailed Description}
User-\/level library to wrap thread-\/synch kernel-\/level functionalities. 

Funcions will operate through two types of structures\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{structT__THREAD__SYNCH}{T\+\_\+\+T\+H\+R\+E\+A\+D\+\_\+\+S\+Y\+N\+CH}}\+: handles request to the main synch device file
\item \mbox{\hyperlink{structT__THREAD__GROUP}{T\+\_\+\+T\+H\+R\+E\+A\+D\+\_\+\+G\+R\+O\+UP}}\+: handles request to a group device file 
\end{DoxyItemize}
